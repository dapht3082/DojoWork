.click(), .hide(), and .show()

The .click function 'listens' for someone to click the element it is attached to, then executes the code passed to it. The .hide and .show functions hide and show an element, respectively.

.toggle

The .toggle() function can be used for a similar effect, but with different default animations and a shorter syntax

.slideUp and .slideDown

Once again, the .slide___ pair functions similarly.

.fadeIn() and .fadeOut()

.fadeIn and .fadeOut work just like .show() and .hide().

.addClass() and .removeClass()

Sadly, .addClass() won't make something less wild and savage.

Instead, .addClass() and .removeClass() adds or removes the passed class to the object it effects.

.before()

.before() inserts whatever it is passed into the position before the element it is assigned to.

.append() and .prepend()

.append( and .prepend() are similar to .before() and .after(), but insert the item into the element selected, rather than adjacent to the element.
